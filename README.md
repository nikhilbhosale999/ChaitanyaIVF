# Shree Chaitanya IVF Center Website

This repository contains the front-end source code for the Shree Chaitanya IVF Center website. The project is designed to be modern, fully responsive, and medical-grade, following premium UI/UX principles.

## Features

- **Modern UI/UX:** Clean, minimalistic aesthetic with ample whitespace, a trustworthy color palette (Deep Teal/Royal Blue, Maternity Pink, Slate), and glassmorphism elements.
- **Fully Responsive:** Mobile-first approach ensuring the site looks and functions perfectly across all devices (desktops, tablets, and mobile phones).
- **Smooth Animations:** Integrated scroll-reveal effects (`IntersectionObserver`), sticky header, and subtle hero image parallax for an engaging user experience.
- **Accessible & Fast:** Written in pure HTML, CSS, and Vanilla JavaScript with no heavy frameworks, ensuring fast loading times.
- **Lead Generation:** Strategically placed, high-contrast CTA buttons and a floating contact form designed to maximize conversions.

## File Structure

- `index.html`: The main HTML document containing the structure and content of the website.
- `style.css`: The stylesheet defining the visual appearance, layout grids, animations, and mobile responsiveness.
- `script.js`: Vanilla JavaScript handling interactivity such as scroll animations, the mobile hamburger menu, and accordion toggles.
- `assets/images/`: Directory containing high-quality, AI-generated images used throughout the site (Hero image, doctors, etc.).

## Technologies Used

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6)
- Fonts: Google Fonts (Plus Jakarta Sans, Inter)
- Icons: Phosphor Icons (via CDN)

## Setup & Usage

To view the website locally, no build process or server is required. 
Simply open the `index.html` file in any modern web browser.

1. Clone or download the repository.
2. Double-click `index.html` to open it in Chrome, Safari, Firefox, or Edge.

## Customization

- **Colors & Branding:** All main colors are defined as CSS variables at the top of `style.css` inside the `:root` pseudo-class. Change these hex codes to update the theme instantly.
- **Typography:** Fonts are imported at the very top of `style.css`. You can change the font families in the CSS variables or body/heading selectors.
